  • Adaptador puente

    El adaptador puente (del inglés: bridged adapter) es aquel que crea un puente entre una máquina virtual y la red a la que está conectada la máquina anfitriona, haciendo parecer que está conectada físicamente a dicha red a través de un cable.

  • Cortafuegos

    El cortafuegos (del inglés: firewall) es un tipo de hardware o software que se encarga de bloquear el acceso no autorizado, usualmente desde internet, bloqueando así aquellos que no cumplen con los criterios de seguridad especificados.

  • DHCP

    DHCP (del inglés: Dynamic Host Configuration Protocol) es un protocolo de red mediante el cual un servidor DHCP asigna dinámicamente una dirección IP y otros parámetros de configuración de red a todos aquellos dispositivos en su red que se lo soliciten.

  • Dirección APIPA

    La dirección APIPA (del inglés: Automatic Private Internet Protocol Addressing) es un protocolo que utilizan los sistemas operativos para obtener la configuración de red cuando está ajustado para obtener una dirección IP de manera dinámica y, al iniciar, este no encuentra un servidor DHCP. Estas direcciones están en el rango 169.254.0.0/24.

  • Dirección MAC

    La dirección MAC (del inglés: Media Access Control) es un identificador de 6 parejas de caracteres hexadecimales (48 bits) que se corresponde de forma única con una tarjeta de red. Están reguladas por el IEEE para asegurarse de que no se puedan asignar dos direcciones iguales a diferentes dispositivos. Hay varias listas, las cuales pueden consultarse aquí.

  • DNS

    DNS (del inglés: Domain Name System) es un sistema jerárquico que asocia nombres de dominio con direcciones IP, con el propósito de resolver los nombres de las URLs, y viceversa.

  • Puerta de enlace

    La puerta de enlace (del inglés: gateway) es aquel dispositivo dentro de dos o más redes en el que los dispositivos de ambas redes delegan sus paquetes con el fin de que estos puedan ir de una red a otra.

  • Red interna

    La red interna (del inglés: internal network) es aquella red privada que sirve para aislar a una máquina o conjunto de estas del exterior, y también de la máquina anfitriona. Pueden comunicarse entre sí, siempre y cuando el nombre de la red sea el mismo en todas ellas.

  • Router

    El enrutador (del inglés: router) es un dispositivo de la capa de red del modelo ISO/OSI que permite intercomunicar una red privada con la red pública, internet, estableciendo una ruta para cada paquete.

  • Modelo ISO/OSI

    El modelo ISO/OSI (del inglés: Open System Interconnection) es un estándar teórico para los protocolos de red, con el objetivo de que sistemas de fabricantes distintos puedan intercambiar información. Se divide en varias capas: física, de enlace, de red, de transporte, de sesión, de presentación y de aplicación.
